## gray-matter 설치

MDX 파일의 frontmatter를 파싱하기 위해 gray-matter를 설치합니다:

```bash
npm install gray-matter
```

## Post 인터페이스 정의

`src/lib/posts.ts`에서 Post 타입을 정의합니다:

```typescript
export interface Post {
  slug: string;
  title: string;
  date: string;
  tags: string[];
  content: string;
  excerpt?: string;
}
```

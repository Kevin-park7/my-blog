# TypeScript 기초 완벽 가이드

## 개요
TypeScript는 JavaScript에 정적 타입을 추가한 언어입니다. 이 가이드에서는 TypeScript의 기본을 배워봅시다.

## 주요 개념

### 1. 기본 타입
```typescript
const name: string = "켈빈";
const age: number = 27;
const isActive: boolean = true;
```

### 2. 인터페이스
```typescript
interface User {
  id: number;
  name: string;
  email: string;
}
```

### 3. 제네릭
```typescript
function getFirstElement<T>(arr: T[]): T {
  return arr[0];
}
```

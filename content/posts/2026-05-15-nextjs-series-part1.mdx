# Next.js 블로그 만들기 - Part 1: 프로젝트 시작

이 시리즈에서는 Next.js를 활용하여 블로그를 만드는 과정을 단계별로 설명합니다.

## 프로젝트 생성

먼저 Next.js 프로젝트를 생성합니다:

```bash
npx create-next-app@latest my-blog
```

## 디렉토리 구조

```
my-blog/
├── app/
├── content/posts/
├── src/
│   ├── components/
│   ├── lib/
│   └── styles/
└── package.json
```

다음 파트에서는 포스트 시스템을 구축하겠습니다.

## 구현 팁

### 게임 추가
```bash
# 1. components/games/ 에 새 게임 컴포넌트
# 2. app/games/page.tsx 에 등록
# 3. styles/globals.css 에 스타일 추가
# 4. 리더보드 통합 (localStorage)
```

### Firebase 통합
```bash
npm install firebase
# Firebase Console > 프로젝트 생성
# .env.local 에 설정 추가
```

### AI 기능 (Claude API)
```bash
npm install @anthropic-ai/sdk
# ANTHROPIC_API_KEY 환경 변수 설정
# API 라우트 생성: app/api/ai/summarize
```

### Discord 봇
```bash
npm install discord.js
# Discord Developer Portal > 봇 생성
# 토큰 .env.local 에 추가
```
